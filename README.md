## Building

### [CalenFit](https://calenfit.com) — AI fitness app, shipped solo · live on iOS & Android

An agentic two-LLM pipeline (interpreter + parser, concurrent) that turns free-text workout notes into structured data, with confidence-gated human-in-the-loop validation and recommendations grounded in a curated knowledge base through deterministic structured retrieval. Runs on a layered reliability system — a single LLM client with retry, malformed-output detection, and model fallback; tail-latency handling on a measured threshold; Sentry observability; and a cost/accuracy/latency benchmark that cut inference cost ~2×.
